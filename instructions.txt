Installa NVM
    questo installa NPM e Node.js

Una sola volta all'inizio, nella cartella principale del progetto, esegui i comandi:
	npm init -y
	npm install express socket.io

Fai partire il server con il comando:
	node server.js

Installa Ngrok
Configura Ngrok (vedi TOKEN dal tuo account di Ngrok)
    ngrok config add-authtoken TOKEN
Poi crea link per internet con:
    ngrok http 3000

In un browser connettiti a
    http://localhost:3000
    o al link fornito da Ngrok
